# inventory-management-system

Simple inventory system to manage products and orders

# Necessary services

MongoDB - you can either install it locally or run in a Docker container

# How to run

- run MongoDB
- run `yarn install`
- copy .env.example to .env and provide all required data
- run in watch mode with `npm run start:dev`

# How to run tests

- run MongoDB
- run `yarn install`
- copy .env.example to .env and provide all required data
- run in watch mode with `npm run test`
